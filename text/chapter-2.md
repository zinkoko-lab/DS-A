# 🧠 **Step 2：ループと再帰（アルゴリズムの基礎）**

---

# ✨ 1. **ループの本質**

ループとは

> **「同じ処理を、条件を満たす限り何度も繰り返す仕組み」**

つまり、

-   処理を繰り返す
-   終わる条件がある
-   変化する状態（カウンタ）がある

これだけです。

---

# ✨ 2. **再帰の本質**

再帰とは

> **「関数が自分自身を呼び出して問題を小さくしていく技法」**

ポイントは 3 つ：

1. **分解**（問題を小さくする）
2. **再帰呼び出し**（自分を呼ぶ）
3. **終端条件**（ここで終了！）

終端条件がないと “無限ループ” になるので注意。

---

# 🔹 まずは擬似言語で理解しよう

## ■ ループ（1〜5 を表示）

```pseudo
i = 1
WHILE i <= 5:
    print(i)
    i = i + 1
```

---

## ■ 再帰（1〜5 を表示）

```pseudo
FUNCTION show(n):
    IF n > 5:
        RETURN
    print(n)
    show(n + 1)
```

これだけで “再帰の本質” を理解できます。

# 🐍 **3. Python で理解（初心者向け・読みやすい）**

---

## ■ ループ（while）

```python
i = 1
while i <= 5:
    print(i)
    i += 1
```

## ■ 再帰

```python
def show(n):
    if n > 5:        # 終端条件
        return
    print(n)
    show(n + 1)      # 問題を小さくして再帰
```

# 🟨 **4. JavaScript（React の母語）**

---

## ■ ループ（while）

```javascript
let i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
```

## ■ 再帰

```javascript
function show(n) {
    if (n > 5) {
        return;
    }
    console.log(n);
    show(n + 1);
}
```

# 🟦 **5. C 言語（再帰のメモリ構造がわかる）**

---

## ■ ループ（while）

```c
#include <stdio.h>

int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}
```

---

## ■ 再帰

```c
#include <stdio.h>

void show(int n) {
    if (n > 5) {        // 終端条件
        return;
    }
    printf("%d\n", n);
    show(n + 1);        // 再帰呼び出し
}

int main() {
    show(1);
    return 0;
}
```

---

---

# 🔥 **ループ vs 再帰**

|            | ループ                 | 再帰                 |
| ---------- | ---------------------- | -------------------- |
| 書き方     | while, for             | 関数を自分で呼ぶ     |
| 終わる条件 | 条件式                 | 終端条件             |
| 読みやすさ | 初心者向け             | 慣れると強力         |
| メリット   | コードが軽い           | 分解しやすく論理的   |
| デメリット | 大きい問題は書きにくい | メモリ消費（特に C） |
